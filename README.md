# Axiora v0.1.0-alpha

## Overview

This is the first functional internal MVP milestone for Axiora Decision Studio.

## Completed

- Authentication (Email & Password)
- Protected Routes
- GitHub Integration
- Supabase Integration
- Decision Board CRUD
- Customer Validation Evidence Management
- Evidence Uploads
- Database Persistence
- Row Level Security (RLS)

## Architecture

- React + TypeScript
- Supabase Backend
- PostgreSQL
- GitHub Version Control
- Lovable Development
- Claude Engineering Reviews
- ChatGPT Product & Architecture Guidance

## Notes

This is an internal development release and is not intended for production use.

The next milestone (v0.2.0-alpha) will focus on implementing the remaining four assessment dimensions using the reusable evidence engine.
